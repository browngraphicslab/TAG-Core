move

(
  • res
  • leftbound
  • rightbound
  • displayIn
  • displayOut
)

Defined in js/TAG/tourauthoring/TAG.TourAuthoring.Display.js:1237

Moves display to an absolute position given in res currentDisplay, offset, loc should be set, see _initSVG / _displayMousedown for details Currently three different types of move, determined by loc variable

  1. loc === 'main': translates the entire display, preserves length
  2. loc === 'fade-in' or 'fade-out': drags only fade-in / fade-out region, start of other fade and lengths of fades remaines fixed, length of main area changes

Parameters:

  • res Event

    event from makeManipulable, onManipulate

  • leftbound Time

    leftmost position display can move to in seconds (not required)

  • rightbound Time

    rightmost position display can move to in seconds (not required)

  • displayIn Time

    the position of a nearby display in the previous trac

  • displayOut Time

    the position of a nearby display in the next track

TAG.TourAuthoring.InkAuthoring Class

Defined in: js/TAG/tourauthoring/TAG.TourAuthoring.InkAuthoring.js:16

Back-end for ink authoring and editing. Instances are created in ComponentControls and InkES. In ComponentControls, we need the ability to author and edit inks. In InkES, we need them to follow artworks. Uses the RaphaelJS library for svg manipulation.
